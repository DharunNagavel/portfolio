Projects

Ableed

Ableed is an assistive learning platform designed to support individuals in understanding Braille Literacy, Lip Reading, and Sign Language. The goal of this project is to bridge communication gaps with the blind, deaf, and mute communities through a user-friendly and educational interface.

This platform includes structured learning modules, quizzes, and accessible content, making it easier for users to navigate and absorb concepts visually and audibly. The backend handles user registration and login functionality securely using PHP and MySQL, while the frontend ensures clarity with a modern layout powered by Bootstrap.
